# Vision Board 스타일 미리보기

## 변경 전 (현재 Recommendations 페이지)
- CareerCard: 텍스트 기반 카드 형식
- 설명, 추천 이유, 스킬 등이 리스트 형태로 표시
- 배경: glass-hero 스타일 (반투명 배경)

## 변경 후 (Vision Board 이미지 형식)

### 1. 전체 레이아웃
```
┌─────────────────────────────────┐
│  [배경: 그라데이션 + 패턴 효과]   │
│                                 │
│  ┌─────────────────────────┐   │
│  │  2040                     │   │
│  │  [진로 이름]              │   │
│  │  [역할] @ [회사명]        │   │
│  └─────────────────────────┘   │
│                                 │
│  [설명 텍스트]                  │
│                                 │
│  ┌─────────────────────────┐   │
│  │  "인용구"                │   │
│  └─────────────────────────┘   │
│                                 │
│  🏅 주요 성과                   │
│  • 성과 1                      │
│  • 성과 2                      │
│                                 │
│  📍 성장 타임라인               │
│  [2028] [2032] [2040]          │
└─────────────────────────────────┘
```

### 2. 스타일별 색상
- **ID Badge**: 파란색 그라데이션 (#1a365d → #2563eb → #1e40af)
- **Magazine Cover**: 핑크 그라데이션 (#831843 → #ec4899 → #be185d)
- **Achievement**: 골드 그라데이션 (#713f12 → #f59e0b → #d97706)

### 3. 주요 특징
- **이미지 비율**: ID Badge (2:3), Magazine Cover (3:4), Achievement (3:4)
- **최소 높이**: 600px
- **배경 패턴**: radial-gradient로 빛 효과
- **텍스트 그림자**: 제목에 text-shadow 적용
- **인용구 박스**: 반투명 배경 + backdrop-filter
- **타임라인**: 카드 형식으로 컴팩트하게 표시

### 4. 시각적 효과
- 그라데이션 배경 (3단계)
- 배경 패턴 오버레이 (opacity: 0.1)
- 텍스트 그림자
- 반투명 박스 (backdrop-filter: blur)
- 카드형 타임라인

## 적용 위치
- Recommendations 페이지의 CareerCard 컴포넌트
- 각 진로 카드를 클릭하면 vision-board 형식으로 확대 표시
- 또는 카드 자체를 vision-board 스타일로 변경
